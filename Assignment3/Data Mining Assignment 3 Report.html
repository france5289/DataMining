Implementation Detail

PageRank

數學推導(Linear Algebra)

以下透過李弘毅老師的線性代數投影片講解如何利用線性代數解決 PageRank 問題
而程式碼實做也是根據線定代數中的 Power Method 計算 PageRank vector

Definition from Wikepeida:

PageRank 透過計算機率分佈來體現某人隨機點擊某個連結的機率，因此若網站 A 同時被多個網站連結，則網站 A 的 PageRank 值 PR(A) 則會高於其他網站。

而 PageRank 同時也引進了 Random Surfer 的概念，即假設某人在瀏覽器中隨機隨機開啟某些頁面並點擊了某些連結。為了便於理解，這裡假設上網者不斷點擊網頁上的連結直到進入一個沒有外部連結的網頁，此時他會隨機瀏覽其他的網頁（可以與之前的網頁無關）。 ……

假設一網路圖如下:

node 1 的有兩個 inlink，分別為 (3,1) (4,1) ，因為 node3 只有一個 outlink 而 node4 具有兩個 outlink，因此可以先天真的假設 node3 的 user 有 100% 的機率瀏覽 node1 而 node4 具有 50% 的機率瀏覽 node4，
因此可得圖中第一個等式。而其他點也根據類似概念推導出其他三個等式。
接著我們將等式利用 matirx 和 column vector 表示，則可得出等式 Ax=xAx=x，此時所求的 PageRank vector xx 即為 matrix AA 的 eigenvalue 為 1 所對應的 eigenvector。

因為矩陣 A 為 column stochastic 因此必有一 eigenvalue λ=1λ=1
A 可透過將 adjacency matrix 進行 column normalized (one-norm) 求得

然而，若只單純透過 cloumn normalized 求得 A 並計算 eigenvalue 為 1 的 eigenvector，可能因為其 eigenspace 的 dimension 大於 1 ， 使得 PageRank 沒有唯一解。因此實際上我們必須透過以公式，確保 PageRank 有唯一解

M=(1−d)A+dSM=(1−d)A+dS

其中 dd 為 damping factor，表示 user 存取頁面後會繼續存取該頁面所 link 的頁面之機率，而 SS 是一矩陣，其維度與 AA 相同均為 n×nn×n，且其 entries 為 1n1n。如此一來 MM 即為一 positive column stochastic 矩陣，因此其 λ=1λ=1 所對應的 eigenvector 必存在，且 eigenspace 維度必為 1，所以所求的 PageRank vector 必唯一。
而因為矩陣 MM 的維度與網路中的網頁個數成正相關，因此若使用特徵方程式找 eigenvector，當最高項次數大於 4 即不存在公式解，所以我們改用 Power Method 。

假設方陣 M 最大的特徵值唯一存在，
Power Iteration:
Find x∗x∗ such that x∗=Mx∗,||x∗||1=1x∗=Mx∗,||x∗||1=1

因此我們假設初始向量 x0,||x0||1=1x0,||x0||1=1，並不斷重複計算x1,x2,…,xkx1,x2,…,xk 直到收斂，則 xkxk 即為解。

程式碼實作

Pseudo code:













A : AdjMatrix with column one-norm normalization
S : NxN matrix S(i,j) = 1/N
M = (1-d)A + dS
criteria = 0.01
d = 0.15 (damping factor)
x = Nx1 column vector with one-norm normalization

while True:
    temp = x
    x = matmul(M,x)
    if norm(x - temp) <= criteria:
        return x

HITS

數學推導(Linear Algebra)

HITS 演算法會計算每個網站的 Authority 與 Hub 值，一個網站的 inlink 越多其 Authority 值越高，而其 outlink 越多則 Hub 值越高。
假設網路圖如下:

可得其 adjacency matrix

A=⎡⎣⎢⎢⎢0001000011000010⎤⎦⎥⎥⎥A=[0010001000011000]

接著我們初始化 AA 的 Authority vector aa 與 Hub vector hh
a=[1,1,1,1]Tb=[1,1,1,1]Ta=[1,1,1,1]Tb=[1,1,1,1]T

則此圖的 Authority 與 Hub 可由以下公式推得
a=AThh=Aaa=AThh=Aa

接著將 hh 替換成 AaAa，aa 替換成 AThATh分別帶回上述公式，則可得
a=ATAa=Mah=AATh=Naa=ATAa=Mah=AATh=Na

其中方陣 MM 與 NN 為實對稱，其絕對值對大的 eigenvalue 必存在且唯一，因此我們可使用 Power Method 計算 Authority 與 Hub vector，實作 HITS 演算法。

程式碼實作

Pseudo code:





















init a : Nx1 vector a(i,1) = 1
init h : Nx1 vector h(i,1) = 1
init Au : NxN matrix Au = matmul(A.T, A)
init Hu : NxN matrix Hu = matmul(A, A.T)
init criteria = 0.01
## Find authority
while True:
    prev = a
    a = matmul(Au, a)
    a <- normalize a with one-norm
    if norm(a - prev) <= criteria:
        break

while True:
    prev = h
    h = matmul(Hu, h)
    h <- normalize h with one-norm
    if norm(h-prev) <= criteria
        break
return a, h

SimRank

數學推導

SimRank 透過計算兩點 aa 與 bb 之間的 in-neighbor 相似性來判斷兩點的相似性，因次若 aa 與 bb 的 in-neighbor 相同的越多，則兩點相似性越高。
SimRank 計算兩點相似性的公式如下：

S(a,b)=1 ,if a = b S(a,b)=0 , if in(a) = 0 or in(b) = 0S(a,b)=C|I(a)||I(b)|∑i=1|I(a)|∑j=1|I(b)|S(Ii(a),Ij(b))S(a,b)=1 ,if a = b S(a,b)=0 , if in(a) = 0 or in(b) = 0S(a,b)=C|I(a)||I(b)|∑i=1|I(a)|∑j=1|I(b)|S(Ii(a),Ij(b))

其中 CC 為 decay factor，0<C<10<C<1。
上述的公式可轉換成矩陣表示法：
S0=InSk=cWTSk−1W+In−Diag(diag(cWTSk−1W))S0=InSk=cWTSk−1W+In−Diag(diag(cWTSk−1W))

其中矩陣 WW 為 column normalized adjacency matrix，Diag(diag(cWTSk−1W)Diag(diag(cWTSk−1W) 則為一 diagonal matirx，其對角項為 cWTSk−1WcWTSk−1W 的對角項。

Pseudo code












init Q : adjacency matrix with column normalization(one-norm)
init S : identity Matrix
init I : identity Matrix
init c = 0.5 (decay factor)
init criteria = 0.01
while True:
    prev = S
    S = c * Q @ S @ Q_T 
    S = S + I - Diag(diag(S))
    if norm(S - prev) <= criteria:
        return S
